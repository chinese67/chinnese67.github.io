หมวดคำศัพท์อาหาร

1.饭 (fàn)

แปลว่า ข้าว

ตัวอย่างการใช้คำศัพท์

A: 你喜欢吃什么饭？ (Nǐ xǐhuān chī shénme fàn?) = คุณชอบกินข้าวอะไร?
B: 我喜欢吃炒饭。 (Wǒ xǐhuān chī chǎofàn.) = ฉันชอบกินข้าวผัด

2. 面条 (miàntiáo)

แปลว่า เส้นก๋วยเตี๋ยว

ตัวอย่างการใช้คำศัพท์

A: 今天午饭吃什么？ (Jīntiān wǔfàn chī shénme?) = วันนี้มื้อกลางวันกินอะไร?
B: 吃面条吧！ (Chī miàntiáo ba!) = กินก๋วยเตี๋ยวกันเถอะ!

3. 面包 (miànbāo)

แปลว่า ขนมปัง

ตัวอย่างการใช้คำศัพท์
>

A: 早餐吃什么？ (Zǎocān chī shénme?) = มื้อเช้ากินอะไร?
B: 我吃面包和牛奶。 (Wǒ chī miànbāo hé niúnǎi.) = ฉันกินขนมปังกับนม

4.牛肉 (niúròu)

แปลว่า เนื้อวัว

ตัวอย่างการใช้คำศัพท์

A: 你喜欢吃牛肉吗？ (Nǐ xǐhuān chī niúròu ma?) = คุณชอบกินเนื้อวัวไหม?
B: 我很喜欢。 (Wǒ hěn xǐhuān.) = ฉันชอบมาก

5. 猪肉 (zhūròu)

แปลว่า เนื้อหมู

ตัวอย่างการใช้คำศัพท์

A: 这道菜是猪肉吗？ (Zhè dào cài shì zhūròu ma?) = เมนูนี้คือเนื้อหมูใช่ไหม?
B: 是的，很好吃。 (Shì de, hěn hǎo chī.) = ใช่ค่ะ/ครับ อร่อยมาก

6. 鸡肉 (jīròu)

แปลว่า เนื้อไก่

ตัวอย่างการใช้คำศัพท์

A: 你喜欢吃鸡肉还是牛肉？ (Nǐ xǐhuān chī jīròu háishì niúròu?) = คุณชอบกินเนื้อไก่หรือเนื้อวัว?
B: 我喜欢吃鸡肉。 (Wǒ xǐhuān chī jīròu.) = ฉันชอบกินเนื้อไก่

7. 鱼 (yú)

แปลว่า ปลา

ตัวอย่างการใช้คำศัพท์

A: 这道菜有鱼吗？ (Zhè dào cài yǒu yú ma?) = อาหารจานนี้มีปลาไหม?
B: 有的。 (Yǒu de.) = มีครับ

8.水果 (shuǐguǒ)

แปลว่า ผลไม้

ตัวอย่างการใช้คำศัพท์

A: 你喜欢吃什么水果？ (Nǐ xǐhuān chī shénme shuǐguǒ?) = คุณชอบกินผลไม้อะไร?
B: 我喜欢吃苹果。 (Wǒ xǐhuān chī píngguǒ.) = ฉันชอบกินแอปเปิ้ล

9. 蔬菜 (shūcài)

แปลว่า ผัก

ตัวอย่างการใช้คำศัพท์

A: 今天的蔬菜很新鲜。 (Jīntiān de shūcài hěn xīnxiān.) = ผักวันนี้สดมาก
B: 是啊，多吃点！ (Shì a, duō chī diǎn!) = ใช่ค่ะ/ครับ กินเยอะๆ นะ

10. 汤 (tāng)

แปลว่า ซุป

ตัวอย่างการใช้คำศัพท์

A: 这碗汤是什么做的？ (Zhè wǎn tāng shì shénme zuò de?) = ซุปชามนี้ทำจากอะไร?
B: 是鸡汤。 (Shì jī tāng.) = ซุปไก่

11. 米饭 (mǐfàn)

แปลว่า ข้าวสวย

ตัวอย่างการใช้คำศัพท์

A: 你喜欢吃什么？ (Nǐ xǐhuān chī shénme?) = คุณชอบกินอะไร?
B: 我喜欢吃米饭和蔬菜。 (Wǒ xǐhuān chī mǐfàn) = ฉันชอบกินข้าวสวย

12. 粥 (zhōu)

แปลว่า โจ๊ก

ตัวอย่างการใช้คำศัพท์

A: 早饭想吃粥还是面包？ (Zǎofàn xiǎng chī zhōu háishì miànbāo?) = มื้อเช้าอยากกินโจ๊กหรือขนมปัง ?
B: 粥，谢谢。 (Zhōu, xièxiè.) = โจ๊ก ขอบคุณ

13. 饺子 (jiǎozi)

แปลว่า เกี๊ยว

ตัวอย่างการใช้คำศัพท์

A: 你想吃点儿什么吗？ (Nǐ xiǎng chī diǎnr shénme ma?) = คุณอยากกินอะไรไหม?
B: 我想吃饺子。 (Wǒ xiǎng chī jiǎozi.) = ฉันอยากกินเกี๊ยว

14. 蛋糕 (dàngāo)

แปลว่า เค้ก

ตัวอย่างการใช้คำศัพท์

A: 你吃过蛋糕吗？ (Nǐ chī guò dàngāo ma?) = คุณเคยกินเค้กไหม?
B: 吃过，很好吃。 (Chī guò, hěn hǎochī.) = เคยกินแล้ว อร่อยมาก

15. 冰淇淋 (bīngqílín)

แปลว่า ไอศกรีม

ตัวอย่างการใช้คำศัพท์

A: 夏天吃冰淇淋真舒服。 (Xiàtiān chī bīngqílín zhēn shūfú.) = กินไอศกรีมหน้าร้อนมันสดชื่นจริงๆ
B: 是啊，我也喜欢。 (Shì a, wǒ yě xǐhuān.) = ใช่ค่ะ/ครับ ฉันก็ชอบ

16. 沙拉 (shālā)

แปลว่า สลัด

ตัวอย่างการใช้คำศัพท์

A: 今天午饭有沙拉吗？ (Jīntiān wǔfàn yǒu shālā ma?) = วันนี้มื้อกลางวันมีสลัดไหม?
B: 有，很健康！ (Yǒu, hěn jiànkāng!) = มีค่ะ/ครับ สุขภาพดีมาก

17. 煎饼 (jiānbǐng)

แปลว่า แพนเค้ก

ตัวอย่างการใช้คำศัพท์

A: 煎饼好吃吗？ (Jiānbǐng hǎo chī ma?) = แพนเค้กอร่อยไหม?
B: 非常好吃！ (Fēicháng hǎo chī!) = อร่อยมาก!

18. 火锅 (huǒguō)

แปลว่า หม้อไฟ

ตัวอย่างการใช้คำศัพท์

A: 你吃过火锅吗？ (Nǐ chī guò huǒguō ma?) = คุณเคยกินหม้อไฟไหม?
B: 吃过，味道很好。 (Chī guò, wèidào hěn hǎo.) = เคยกินแล้ว รสชาติอร่อยมาก

19.酒 (jiǔ)

แปลว่า เหล้า

ตัวอย่างการใช้คำศัพท์

A: 你想喝点儿酒吗？ (Nǐ xiǎng hē diǎnr jiǔ ma?) = คุณอยากดื่มเหล้าบ้างไหม?
B: 不想，谢谢。 (Bù xiǎng, xièxiè.) = ไม่อยาก ขอบคุณ

20. 茶 (chá)

แปลว่า ชา

ตัวอย่างการใช้คำศัพท์

A: 你喝茶吗？ (Nǐ hē chá ma?) = คุณดื่มชาไหม?
B: 是的，我每天喝茶。 (Shì de, wǒ měitiān hē chá.) = ใช่ ฉันดื่มชาทุกวัน
