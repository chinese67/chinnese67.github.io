หมวดคำศัพท์คำทักทาย

1. 你好 (Nǐ hǎo)

แปลว่า สวัสดี

ตัวอย่างการใช้คำศัพท์

A: 你好！ (Nǐ hǎo!) = สวัสดี
B: 你好！今天怎么样？ (Nǐ hǎo! Jīntiān zěnme yàng?) = สวัสดี วันนี้เป็นอย่างไรบ้าง?

2. 早上好 (Zǎoshang hǎo)

แปลว่า สวัสดีตอนเช้า

ตัวอย่างการใช้คำศัพท์

A: 早上好！ (Zǎoshang hǎo!) = สวัสดีตอนเช้า
B: 早上好！今天工作忙吗？ (Zǎoshang hǎo! Jīntiān gōngzuò máng ma?) = สวัสดีตอนเช้า วันนี้งานยุ่งไหม?

3. 晚上好 (Wǎnshàng hǎo)

แปลว่า สวัสดีตอนเย็น

ตัวอย่างการใช้คำศัพท์
>

A: 晚上好！你今天过得怎么样？ (Wǎnshàng hǎo! Nǐ jīntiān guò de zěnme yàng?) = สวัสดีตอนเย็น วันนี้เป็นอย่างไรบ้าง?
B: 很好，谢谢！ (Hěn hǎo, xièxiè!) = ดีมาก ขอบคุณ

4. 再见 (Zàijiàn)

แปลว่า ลาก่อน

ตัวอย่างการใช้คำศัพท์

A: 我得走了，再见！ (Wǒ děi zǒu le, zàijiàn!) = ฉันต้องไปแล้ว ลาก่อน!
B: 再见！祝你一路平安！ (Zàijiàn! Zhù nǐ yīlù píng ān) = ลาก่อน ขอให้เดินทางปลอดภัย

5. 谢谢 (Xièxiè)

แปลว่า ขอบคุณ

ตัวอย่างการใช้คำศัพท์

A: 这是给你的礼物。 (Zhè shì gěi nǐ de lǐwù.) = นี่เป็นของขวัญสำหรับคุณ
B: 谢谢！我很喜欢。 (Xièxiè! Wǒ hěn xǐhuān.) = ขอบคุณ! ฉันชอบมาก

6. 不客气 (Bù kèqì)

แปลว่า ไม่เป็นไร

ตัวอย่างการใช้คำศัพท์

A: 谢谢你帮了我的忙。 (Xièxiè nǐ bāng le wǒ de máng.) = ขอบคุณที่ช่วยฉัน
B: 不客气，这是小事。 (Bù kèqì, zhè shì xiǎoshì.) = ไม่เป็นไร เรื่องเล็กน้อย

7. 对不起 (Duìbùqǐ)

แปลว่า ขอโทษ

ตัวอย่างการใช้คำศัพท์

A: 对不起，我忘记了约会时间。 (Duìbùqǐ, wǒ wàngjì le yuēhuì shíjiān.) = ขอโทษ ฉันลืมเวลานัด
B: 没关系，下次记得就好。 (Méi guānxi, xià cì jìde jiù hǎo.) = ไม่เป็นไร ครั้งหน้าจำไว้ก็พอ

8. 没关系 (Méi guānxi)

แปลว่า ไม่เป็นไร

ตัวอย่างการใช้คำศัพท์

A: 对不起，打扰你了。 (Duìbùqǐ, dǎrǎo nǐ le.) = ขอโทษที่รบกวนคุณ
B: 没关系，请继续说。 (Méi guānxi, qǐng jìxù shuō.) = ไม่เป็นไร เชิญพูดต่อ

9. 请 (Qǐng)

แปลว่า กรุณา

ตัวอย่างการใช้คำศัพท์

A: 请坐！ (Qǐng zuò!) = เชิญนั่ง
B: 谢谢！ (Xièxiè!) = ขอบคุณ

10. 没问题 (Méi wèntí)

แปลว่า ไม่มีปัญหา

ตัวอย่างการใช้คำศัพท์

A: 我可以用一下你的电脑吗？ (Wǒ kěyǐ yòng yīxià nǐ de diànnǎo ma?) = ฉันขอใช้คอมพิวเตอร์ของคุณได้ไหม?
B: 没问题，请随便用。 (Méi wèntí, qǐng suíbiàn yòng.) = ไม่มีปัญหา เชิญใช้ได้เลย

11. 你好，早上好 (Nǐ hǎo, zǎoshang hǎo)

แปลว่า สวัสดีตอนเช้า

ตัวอย่างการใช้คำศัพท์

A: 你好，早上好！今天的天气真好。 (Nǐ hǎo, zǎoshang hǎo! Jīntiān de tiānqì zhēn hǎo.) = สวัสดีตอนเช้า! วันนี้อากาศดีจริง ๆ
B: 是啊，适合出去散步。 (Shì a, shìhé chūqù sànbù.) = ใช่เลย เหมาะกับการออกไปเดินเล่น

12. 你好吗？ (Nǐ hǎo ma?)

แปลว่า คุณสบายดีไหม?

ตัวอย่างการใช้คำศัพท์

A: 你好吗？最近怎么样？ (Nǐ hǎo ma? Zuìjìn zěnme yàng?) = คุณสบายดีไหม? ช่วงนี้เป็นอย่างไรบ้าง?
B: 我很好，谢谢你关心！ (Wǒ hěn hǎo, xièxiè nǐ guānxīn!) = ฉันสบายดี ขอบคุณที่ถามไถ่!

13. 我很好 (Wǒ hěn hǎo)

แปลว่า ฉันสบายดี

ตัวอย่างการใช้คำศัพท์

A: 最近怎么样？ (Zuìjìn zěnme yàng?) = สบายดีไหมช่วงนี้?
B: 我很好，谢谢你关心！ (Wǒ hěn hǎo, xièxiè nǐ guānxīn!) = ฉันสบายดี ขอบคุณที่ห่วงใย!

14. 你叫什么名字？ (Nǐ jiào shénme míngzì?)

แปลว่า คุณชื่ออะไร?

ตัวอย่างการใช้คำศัพท์

A: 你叫什么名字？ (Nǐ jiào shénme míngzì?) = คุณชื่ออะไร?
B: 我叫丽丽。 (Wǒ jiào Lìlì.) = ฉันชื่อ ลี่ลี่

15. 我叫... (Wǒ jiào...)

แปลว่า ฉันชื่อ

ตัวอย่างการใช้คำศัพท์

A: 你叫什么名字？ (Nǐ jiào shénme míngzì?) = คุณชื่ออะไร?
B: 我叫明明。 (Wǒ jiào Míngmíng.) = ฉันชื่อ หมิงหมิง

16. 你来自哪里？ (Nǐ láizì nǎlǐ?)

แปลว่า คุณมาจากไหน

ตัวอย่างการใช้คำศัพท์

A: 你来自哪里？ (Nǐ láizì nǎlǐ?) = คุณมาจากไหน?
B: 我来自泰国。 (Wǒ láizì Tàiguó.) = ฉันมาจากประเทศไทย

17. 我来自... (Wǒ láizì...)

แปลว่า ฉันมาจาก...

ตัวอย่างการใช้คำศัพท์

A: 你来自哪里？ (Nǐ láizì nǎlǐ?) = คุณมาจากไหน?
B: 我来自中国。 (Wǒ láizì Zhōngguó.) = ฉันมาจากประเทศจีน

18. 请问 (Qǐng wèn)

แปลว่า ขอถามหน่อย

ตัวอย่างการใช้คำศัพท์

A: 请问，这里是火车站吗？ (Qǐng wèn, zhèlǐ shì huǒchēzhàn ma?) = ขอโทษครับ/ค่ะ ที่นี่คือสถานีรถไฟไหม?
B: 是的，往前走就是。 (Shì de, wǎng qián zǒu jiù shì.) = ใช่ครับ/ค่ะ เดินไปข้างหน้าเลยจะถึง

19. 认识你很高兴 (Rènshí nǐ hěn gāoxìng)

แปลว่า ยินดีที่ได้รู้จัก

ตัวอย่างการใช้คำศัพท์

A: 你好，我叫丽丽。 (Nǐ hǎo, wǒ jiào Lìlì.) = สวัสดีค่ะ/ครับ ฉันชื่อ ลี่ลี่
B: 认识你很高兴！ (Rènshí nǐ hěn gāoxìng!) = ยินดีที่ได้รู้จัก!

20. 祝你一天愉快 (Zhù nǐ yītiān yúkuài)

แปลว่า ขอให้คุณมีวันที่ดี

ตัวอย่างการใช้คำศัพท์

A: 我先走了，再见！ (Wǒ xiān zǒu le, zàijiàn!) = ฉันขอตัวก่อนนะ ลาก่อน!
B: 再见！祝你一天愉快！ (Zàijiàn! Zhù nǐ yītiān yúkuài!) = ลาก่อน! ขอให้คุณมีวันที่ดี!
